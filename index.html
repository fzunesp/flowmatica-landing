Frequently Asked Questions

What's the 30-day guarantee?

If you're not satisfied within 30 days, email us for a full refund. You keep the software installer anyway. No questions asked.

Can I really run this on my own hardware?

Yes! Each app is packaged with Docker Compose. Run it on any machine with Docker installed: Windows, Mac, Linux, or even a Raspberry Pi. No cloud accounts required.

Can multiple users access it?

Yes! Install it on a server on your network and everyone connects to it. No per-user fees. Ever. Add 5 users or 500 users - same price.

What if I need updates or support?

Lifetime security updates are included. Download new versions and run them when YOU'RE ready. Email support for the first year included. Community forum forever.

Can I migrate from my current SaaS tools?

Yes! Each app includes CSV import tools. We'll add migration guides for popular platforms as we go. Plus you can export from our apps anytime to CSV, JSON, or PDF.

What if my server crashes?

Everything is stored in a PocketBase SQLite database file. Back it up to anywhere (USB drive, NAS, cloud if you want). Restore takes 30 seconds. It's just a file.

This sounds too good to be true. What's the catch?

No catch. We're building a sustainable business by selling software once, not renting it forever. You pay for value, not recurring access. It's how software used to work before SaaS took over.

What are the minimum system requirements?

Any machine that can run Docker: 2GB RAM minimum (4GB recommended), 500MB disk space per app, any modern CPU. Yes, even a Raspberry Pi 4 works great.
